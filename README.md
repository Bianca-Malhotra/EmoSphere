# 🌟 EmoSphere - AI-Powered Mental Wellness & Mood Tracking
### A modern, interactive platform for emotional health tracking, AI-powered wellness recommendations, and personal growth. Built with React, TypeScript, and Google Gemini AI for intelligent mental wellness support.
---

## ✨ Features

### Core Mood & Wellness Tracking
- **Mood Tracking** - Log daily moods (Happy, Calm, Sad, Anxious, Energized, Tired) with timestamps
- **Mood History** - View past mood entries with detailed visualization
- **Mood Insights** - AI-powered analysis of mood patterns and trends
- **Moodboard** - Visual representation of your emotional journey

### Wellness & Self-Care
- **Wellness Goals** - Set, track, and manage personal wellness objectives
- **Gratitude Journal** - Daily gratitude entries for positive reinforcement
- **Image Journal** - Visual mood and memory journaling with images
- **Breathing Exercises** - Guided relaxation and breathing techniques
- **Wellness Tips** - AI-generated personalized wellness recommendations

### AI & Intelligence
- **AI Analysis** - Google Gemini-powered sentiment analysis
- **Personalized Advice** - Contextual recommendations based on mood and entry history
- **Wellness Voice Agent** - Voice-based interactions for hands-free wellness guidance
- **Daily Quotes** - Inspirational and wellness-focused daily quotes

### Assessment & Monitoring
- **Weekly Vibration Analysis** - Weekly wellness assessment tracking:
  - Weather/environment impact
  - Noise levels
  - Social connection metrics
  - Tension levels
- **Wellness Assessment Modal** - Comprehensive wellness evaluation tool
- **Concentration Hub** - Focus and productivity support tools
- **Daily Activity** - Activity tracking for holistic wellness

### User & Account Management
- **User Authentication** - Secure email-based login/sign-up
- **Premium Subscription** - Unlock advanced features with upgrade option
- **Theme Customization** - Multiple visual themes and color schemes
- **Crisis Support** - Quick access to crisis support resources and hotlines
- **Privacy & Terms** - Clear privacy policy and terms of service

## 📦 Prerequisites

Before you begin, ensure you have:

- **Node.js** (v16 or higher) - [Download](https://nodejs.org/)
- **npm** (v8 or higher) - Comes with Node.js
- **Google Gemini API Key** - [Get one here](https://aistudio.google.com/app/apikey)

---

## 🚀 Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Bianca-Malhotra/EmoSphere.git
   cd EmoSphere
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

---

## ⚙️ Configuration

1. **Create environment file:**
   Create a `.env.local` file in the root directory:
   ```bash
   touch .env.local
   ```

2. **Add your Gemini API Key:**
   ```
   VITE_GEMINI_API_KEY=your_gemini_api_key_here
   ```

   > ⚠️ **Important:** Never commit `.env.local` to version control. It's already in `.gitignore`.

---

## ▶️ Running the App

### Development Mode
```bash
npm run dev
```
The app will be available at `http://localhost:5173`

### Build for Production
```bash
npm run build
```
Compiles TypeScript and creates an optimized production build in the `dist/` folder.

### Preview Production Build
```bash
npm run preview
```
Locally preview the production build before deployment.
